# UIsounds
Made for muOS.

Instructions:

Download the .zip of the pack you want.

Exctract the contents and place them in mnt/mmc/MUOS/theme/ [theme you want to modify] /sound.

Make sure Navigation Sound is set to “Theme”. You find it in: Configuration > Customization > Navigation Sound. You’ll also find the option to enable background music there.

You may want to backup the original sound files, or even duplicate the full theme folder and only modify the copy to make it easy to go back in case you don’t like the results.

2000sTech includes a .ogg that is background music. This one should be placed in …/MUOS/music.
